# J&J Bombas e Equipamentos Hidráulicos

Site institucional + CRM Admin

## Deploy na Vercel

1. Faça upload deste projeto no GitHub
2. Conecte o repositório na Vercel (vercel.com/new)
3. Framework: **Vite**
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Deploy!

## Acesso CRM Admin

- Triplo-clique no copyright do footer
- Senha: `jj2024admin`

## Stack

- React 18 + Vite
- Poppins (Google Fonts)
- CSS-in-JS
- LocalStorage CRM
